# Snowmaking reservoirs revegetation using macrophytes

## Description

This repository contains data and R scripts used in the article:
**"Establishment of aquatic plants in snowmaking reservoirs: field assessment of littoral bioengineering and biodiversity responses"**
Ecological Engineering (under revision)

## Repository structure

* **Data/**
  * `Data for R/`: cleaned datasets used for analyses
  * `Raw data/`: original datasets
 
* **Scripts/**
  * R scripts used for data processing, analysis and figures

## Reproducibility

All analyses can be reproduced using the datasets in `Data/Data for R` and the scripts in `Scripts/`.

To reproduce the results:

1. Clone or download this repository
2. Open the R scripts in the `Script` folder
3. Ensure the working directory corresponds to the root of the repository
4. Run the scripts sequentially
